# ☕ Cozy Planner: An AI-Powered Task Manager

A minimalist, "cafe-vibes" productivity tool that uses Machine Learning to help you focus on what matters most.

## ✨ Key Features
* [cite_start]**AI Prioritization:** Uses a Scikit-learn model to analyze task effort and calculate priority scores.
* [cite_start]**Multi-User Support:** Secure signup/login system with individual private task lists[cite: 43].
* [cite_start]**Cozy Aesthetic:** Custom-built UI featuring glassmorphism effects and a retro pixel-art cafe background[cite: 221, 241].
* [cite_start]**Coffee Indicators:** Visual "coffee cup" levels that change based on task urgency[cite: 159, 177, 403].

## 🛠️ Tech Stack
* [cite_start]**Backend:** Python (Flask)[cite: 2].
* [cite_start]**Database:** SQLite (Relational data storage)[cite: 43].
* [cite_start]**Machine Learning:** TfidfVectorizer & Logistic Regression[cite: 85, 86].
* [cite_start]**Frontend:** HTML5, CSS3, JavaScript (ES6+)[cite: 116, 189, 219].
* **Deployment:** Render & Gunicorn.

## 🚀 How to Run Locally
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run `python backend/app.py`.
